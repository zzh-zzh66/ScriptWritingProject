# 漫剧剧本正文写作指南

本文档为AI生成漫剧的剧本写作指南，所有规则必须严格遵守，确保剧本可直接用于AI漫画生成。

---

## 一、核心原则（最高优先级）

**剧本正文内容是为了AI生成漫剧服务，不是给人看的！AI无法理解抽象描写！**

### 绝对禁止事项（零容忍）
1. ❌ **禁止所有"眼神XX"表达**：眼神冷、眼神锐利、眼神一冷、眼神扫过、眼神没离开、眼神坚定、眼神警惕等
2. ❌ **禁止在台词括号中描写眼神**：如`陆念离（眼神扫过）：...`
3. ❌ **禁止"眼神如XX"比喻**：如眼神如刀、眼神如炬
4. ❌ **禁止声音描写**：声音冷、声音狠、声音发颤、喉间压狠音、声音沙哑等
5. ❌ **禁止心理描写**：心中暗爽、很害怕、心里想等
6. ❌ **禁止情感描述**：气氛紧张、很愤怒、恐惧等
7. ❌ **禁止抽象描述**：很安静、气质清雅、优雅等
8. ❌ **禁止比喻**：像什么、如什么、如诗如画等
9. ❌ **禁止高级修辞**：排比、拟人、夸张等
10. ❌ **禁止形容词和副词修饰动作**：冷冷地、狠狠地、轻轻、快速等

## 二、颜色标记规范

严格按以下定义标注，确保每集覆盖"高光+冲突+钩子"三类内容。

### 1.【绿色】高光场景·金手指·大爽点
**核心含义**：呈现"强爽感"的关键内容

**包括**：
- 金手指激活（系统绑定、能力解锁）
- 高光操作（神品画作引异象）
- 奖励获取（摆烂值到账、召唤权解锁）

**标注要求**：
- 每集至少1处
- 在单集前1/3篇幅至少有一次
- 内容需"视觉化+爽感明确"

**示例**：
```
【绿色】
△半空中浮现淡蓝色透明面板，上面显示"系统绑定成功！新手大礼包激活：苍生笔、修罗剑意！"
```

### 2.【黄色】冲突·矛盾·纷争触发
**核心含义**：推动剧情张力的"对抗性内容"

**包括**：
- 刺杀、对峙、质疑
- 危机爆发（影卫劈杀、眼线被发现、权贵挑衅）

**标注要求**：
- 每集至少1-2处
- 需与主线相关（如围绕"对抗太子"展开）
- 冲突过程需"动作+对话"结合

**示例**：
```
【黄色】
△为首刺客刀刃距眉心半寸，陆念离胸口涌出鲜血，染红白袍。
```

### 3.【蓝色】铺垫·钩子·悬念
**核心含义**：为后续剧情埋线索的"引导性内容"

**包括**：
- 伏笔埋设（"太子与国舅密谈未查明"）
- 结尾钩子（"陆念离OS'朱峰，这笔账慢慢算'"）
- 未知悬念（"霍仙儿留下'赢了再说事'的神秘承诺"）

**标注要求**：
- 每集结尾必须至少1处"强钩子"
- 指向明确后续剧情
- 单集内可补充1处铺垫型内容

**示例**：
```
【蓝色】
△半空中淡蓝色透明面板弹出，面板显示"检测到暗黑血咒气息，危险等级：★★★"
```

## 三、新写作技巧

### 1. 金色字体浮动描写（人物介绍）

**用途**：配角初次登场时，用金色字体简单交代人物身份背景和姓名

**出场顺序（必须遵守）**：
1. **先有人物外貌描写**：确保AI能生成人物画面
2. **再有悬浮字介绍**：在人物画面基础上添加介绍

**格式**：
```
△人物外貌描写（穿着、外貌特征）
△金色字体，竖向，悬浮在[人物名]旁边，显示：人物名：身份背景
```

**正确示例**：
```
△陆长乐站在宴席旁，穿黑色劲装，腰间佩剑，眉头微皱。
△金色字体，竖向，悬浮在陆长乐旁边，显示：陆长乐：镇北王长女，镇北军统领
```

```
△为首黑影穿夜行衣，蒙面，手持寒刃。
△金色字体，竖向，悬浮在为首黑影旁边，显示：七杀楼刺客首领：太子雇佣的杀手
```

**错误示例**：
```
❌ △金色字体，竖向，悬浮在人物旁边，显示：陆长乐：镇北王长女，镇北军统领
（问题：没有先描写人物外貌，且位置不精确）
```

**注意事项**：
- 用 `△` 符号
- 不要出现"想要"等意图词
- 只出现名字和背景
- 所有配角都要，但无名无信的不需要
- 大场面战争，小罗罗，路人等不需要
- **位置必须精确**：写明"悬浮在XXX旁边"，不能写"悬浮在人物旁边"

### 2. 场景标注

**格式**：
```
△金色字体，横向，悬浮在场景上方，显示：地点名
```

**用途**：每个新场景开头使用，让观众清楚知道当前场景位置

**示例**：
```
△金色字体，横向，悬浮在场景上方，显示：大奉北域成安郡
```

### 3. 系统面板可视化

**格式**：
```
△系统面板：可视化描述
```

**用途**：系统转盘、任务栏等需要描写可视化效果，不只是声音

**示例**：
```
△系统面板：半空中浮现淡蓝色透明面板，上面写着"主线任务：保命护姐，当前进度0/10"
声音提示：叮（系统提示音）
系统：主线任务已激活！
```

**注意事项**：
- 必须描写面板的可视化效果（颜色、位置、形状）
- 不能只写文字，要描述面板本身

### 4. 旁白规则

**格式**：简单直白，清楚交代信息

**用途**：交代世界观、背景信息，帮助观众理解剧情

**示例**：
```
△金色字体，竖向，悬浮在场景上方，显示：大奉北域成安郡
△俯视图：大奉北域成安郡，镇北王府在北，皇宫在南
旁白：大奉皇朝，北域成安郡，镇北王府镇守北域，皇室忌惮兵权，联合七杀楼刺杀陆念离。
```

**注意事项**：
- 旁白要简单直白，不要文邹邹
- 清楚交代世界观信息
- 与场景标注、俯视图配合使用

## 四、描写提示词化（适配AI漫画）

所有场景、动作、人物细节需按"AI可直接生成画面"的逻辑撰写，简单直白，避免文艺化表述。

### 1. 场景描写：明确"时间+地点+物品+状态"

**格式要求**：
```
△时间，地点，物品+状态
```

**正确示例**：
```
△夜，镇北王府世子寝殿，烛火摇晃，桌案摊半幅美人图，苍生笔斜插墨砚。
```

**错误示例**：
- ❌ 寝殿很安静（抽象描述）
- ❌ 烛火如星（比喻）
- ❌ 夜幕降临，镇北王府世子寝殿内烛火摇曳不定（啰嗦）

### 2. 动作描写：拆解"连贯步骤+动态细节"

**格式要求**：
- 动作需拆分为"可镜头捕捉"的步骤
- 补充动态细节增强画面感
- **每个△只包含1-2个核心动作，不超过50字**

**拆分原则**：
- 一个△对应一个漫剧画面
- 复杂动作链必须拆分为多个△
- 每3-5个△必须穿插1-2句对话

**正确示例**：
```
△两刺客一人攻下盘，一人劈肩头，刀刃带风刮动衣摆。
剩余两名刺客（左右分开，挥刀扑来）：这小子藏了实力！用合击术！
△陆念离脚尖点地跳半尺。
△苍生笔连点两下，金色画纹变锁链。
△锁链一端缠左侧刺客手腕，一端勒右侧刺客脖子。
陆念离（收劲）：不堪一击。
```

**错误示例（长描写未拆分）**：
```
❌ △两刺客一人攻下盘，一人劈肩头，刀刃带风刮动衣摆。陆念离脚尖点地跳半尺，苍生笔连点两下，金色画纹变锁链，一端缠左侧刺客手腕，一端勒右侧刺客脖子。
（问题：一个△包含太多动作，AI无法生成单个画面）
```

**正确示例**：
```
△影卫翻身入窗，足尖垫黑布落地，反握寒刃围向床榻。
△陆念离指尖勾住玉画筒，指节因用力泛白，翻身时月白寝衣被刀刃划开长口。
```

**错误示例**：
- ❌ 影卫悄悄进殿准备刺杀（抽象）
- ❌ 他冷冷地踢了踢尸体（形容词修饰）

### 3. 人物细节：聚焦"外貌+简单的表情"

**格式要求**：
- 人物特征需"可视觉识别"
- 表情/状态通过"动作+微表情"呈现

**正确示例（简单直接）**：
```
△张怀安拿卷宗的手发抖，脸色惨白。
```

**错误示例**：
- ❌ 陈蒹葭气质清雅（抽象）
- ❌ 张怀安很害怕（心理描写）

## 五、内容穿插逻辑

核心解决"大篇幅无对话的场景/动作描写"问题，确保每段内容既有画面，又有声音互动。

### 1. 穿插比例
**每3-5句场景/动作描写，必搭配1-2句对话**

**禁止**：连续5句以上纯场景/动作

**正确示例**：
```
△影卫首领挥刀劈向陆念离眉心，寒光擦鼻尖过，锦缎被劈出裂口。（动作）
为首刺客（挥刀再劈）：受死！（对话）
△刀距眉心两寸时，陆念离猛地睁眼，左手撑床翻身，右腿屈膝踹向影卫小腹。（动作）
陆念离（脚掌触甲胄）：朱峰派你们来送死！（对话）
```

### 2. 互动逻辑
- 对话需触发后续动作
- 动作需强化对话情绪

**示例**：
- 百晓通："十万两才给情报！" → 陆念离召出画剑，动作回应
- 刘武："今天废了你！" → 抬脚狠狠踹向叶凌胸口

### 3. 避免堆砌
场景/动作描写以"支撑对话、推动冲突"为目的，避免堆砌无关细节。

---

## 六、音效标注规范

所有拟声词需"单独成段+明确标注"，确保AI能精准生成对应音效。

### 1. 标注格式
```
声音提示：XXX（音效来源说明，可选）
```

**基础格式**：
- 声音提示：当啷（寒刃砸在青砖上）
- 声音提示：咔嚓（骨头断裂声）
- 声音提示：BOOM（符咒爆炸声）

### 2. 位置要求
**紧跟对应动作/场景，不脱节**

**正确示例**：
```
△影卫首领直挺挺倒地，寒刃从手中滑落，砸在青砖上。
声音提示：当啷（寒刃落地声）
```

**禁止**：无对应动作的孤立音效

### 3. 类型限制
仅标注"可视觉关联"的音效（打斗、物品碰撞、爆炸），避免抽象音效。

---

## 七、单集核心要求

确保单集内容饱满、节奏紧凑，符合观众观看预期。

### 1. 高光（【绿色】）
每集至少2处，爽感明确，需是"金手指变现/目标达成"的关键节点。

### 2. 冲突（【黄色】）
每集至少1-2处，张力拉满，需"主动触发+可解决"，解决方式需体现主角能力。

### 3. 钩子（【蓝色】）
每集至少结尾1处，指向后续，需"具体、有期待感"。

### 4. 音效
每集数量不要过多，目的是增强代入感，覆盖"打斗+关键动作"场景。

---

## 八、主线明确化（前三集必须遵守）

前三集是观众理解世界观、人物身份和故事主线的关键时期，必须通过旁白，OS和系统任务提示明确主线。

### 第一集（必须包含）

**1. 利用俯视图+旁白、人物OS，台词等交代背景**
- 世界观：俗世百朝，架空玄幻世界
- 背景：穿越成大奉皇朝，镇北王世子，表面摆废柴纨绔
- 身份：穿越者，心思缜密、权谋无双
- 主线目标：守护姐姐陆长乐，最终一统百朝，终结乱世

**2. 系统任务提示**
在第一集结尾，通过系统任务提示明确阶段性主线目标

### 第二集（必须包含）

**1. 主线推进**
通过台词或系统任务提示显示主线进展

**2. 系统任务提示**
显示当前进度和下一步目标

### 第三集（必须包含）

**1. 主线目标明确**
通过台词明确表达最终目标（一统百朝，终结乱世）

**2. 系统任务提示**
显示当前进度和下一步目标

### OS使用规范
- **使用场景**：仅用于交代信息背景（如第一集交代世界观、背景、身份、主线目标），不要频繁使用
- **禁止事项**：禁止使用OS描写人物心理活动，必须通过动作和对话呈现

## 九、白描原则

所有描写必须简单、直白、清晰明确，确保AI可直接生成画面。

### 动作描写

**格式要求**：
- 简洁直白，使用"主体+动词+对象"结构

**正确示例**：
- ✅ 陆念离踢影卫尸体
- ✅ 陆念离拔剑
- ✅ 陆长乐看向门口

**错误示例**：
- ❌ 陆念离冷冷地踢了踢影卫尸体（形容词修饰）
- ❌ 陆念离眼神锐利，一脚踢飞影卫尸体（眼神描写+比喻）
- ❌ 陆念离狠狠地踹向敌人（副词修饰）

### 场景描写

**格式要求**：
- 白描，简单直白，格式"时间+地点+物品+状态"

**正确示例**：
- ✅ 夜，镇北王府世子寝殿，烛火摇晃
- ✅ 日，皇宫大殿，百官站立

**错误示例**：
- ❌ 烛火在夜风里晃，光投在床榻锦缎上（啰嗦）
- ❌ 烛火如星（比喻）
- ❌ 夜幕降临，镇北王府世子寝殿内烛火摇曳不定（啰嗦）

### 台词后括号内容

**格式要求**：
- 简单直白，只保留核心动作

**正确示例**：
- ✅ （拔剑）
- ✅ （看向那人）
- ✅ （拳头砸桌子）

**错误示例**：
- ❌ （声音发颤）
- ❌ （喉间压狠音）
- ❌ （带哭腔）
- ❌ （眼神里闪过一丝狠厉）
- ❌ （愤怒地）

## 十、禁止眼神描写详解

### 禁止事项清单
- ❌ 眼神冷
- ❌ 眼神锐利
- ❌ 眼神一冷
- ❌ 眼神变得专注
- ❌ 眼神扫过人群
- ❌ 眼神没离开某人
- ❌ 眼神坚定
- ❌ 眼神警惕
- ❌ 眼神频繁瞟
- ❌ 眼神怒
- ❌ 眼神骤亮

### 正确替代方案
使用具体动作：
- ✅ 看向窗外
- ✅ 看向远方
- ✅ 看向某人
- ✅ 转头看向门口
- ✅ 看着楼下
- ✅ 看向皇宫方向
- ✅ 抬眼看向某处
- ✅ 低头
- ✅ 攥紧拳头
- ✅ 咬紧牙关

### 示例对比

❌ **错误示例**：
- 陆念离眼神冷，嘴角勾着懒笑，月白锦袍在夜风里飘。
- 他甩胳膊，眼神转冷，笔尖指向影卫首领。
- 陆念离（眼神坚定）：我会做到的！
- 百晓通猛站起，折扇"唰"打开挡胸口，眼神警惕。

✅ **正确示例**：
- 他甩胳膊，笔尖指向影卫首领。
- 陆念离（看向皇宫方向）：我会做到的！
- 百晓通猛站起，折扇"唰"打开挡胸口。

## 十一、检查清单

生成剧本后必须逐项检查：

### 禁止事项检查
- [ ] 是否有"眼神冷"、"眼神锐利"、"眼神扫过"？→ 全部删除
- [ ] 是否有"声音冷"、"声音狠"、"声音发颤"、"喉间压狠音"？→ 全部删除
- [ ] 台词括号中是否有眼神或声音描写？→ 全部删除
- [ ] 是否有心理描写（心中暗爽、很害怕）？→ 全部删除
- [ ] 是否有情感描述（气氛紧张、很愤怒）？→ 全部删除
- [ ] 是否有抽象描述（很安静、气质清雅）？→ 全部删除
- [ ] 是否有比喻（像什么、如什么）？→ 全部删除
- [ ] 是否有形容词和副词修饰动作？→ 全部删除

### 必须包含检查
